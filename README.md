# 🎬 CineTrack API - Full Stack Movie Management System

This project is a fully containerized (Dockerized) web application developed as a **Computer Engineering student** at Yalova University. It allows users to manage and list their favorite movies seamlessly.

## 🚀 Key Features
* **Backend:** Robust RESTful API built with **Spring Boot (Java)**.
* **Database:** **MySQL 8.0** running within a dedicated Docker container.
* **Frontend:** Clean and responsive UI developed using Vanilla JS + HTML5.
* **DevOps:** Entire infrastructure is managed via `docker-compose` for one-click deployment.
* **Intelligence (Roadmap):** Future integration of a movie recommendation engine based on genres.

## 🛠️ Technology Stack
* **Java 17 & Spring Boot 3**
* **Spring Data JPA & Hibernate**
* **MySQL**
* **Docker & Docker Compose**
* **Maven**
